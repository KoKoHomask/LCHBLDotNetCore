# LCHBLDotNetCore
银行汇率抓取<br />
支持建行、农行、工行、中国银行、邮政储蓄的汇率抓取基础上新增了交通银行、民生银行、招商银行、<br />
北京银行、浦发银行、中信银行、光大银行、华夏银行、广发银行<br />
